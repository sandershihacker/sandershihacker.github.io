My Projects

These are some of the projects that I have worked on.

CMU Summit Club Website

Fall, 2016

During the fall semester of 2016 I redesigned the club website for CMU Summit, and switched to use SCSS instead of CSS to make it easier to maintain. Feel free to click here to visit the site!

Tuxlab

Summer, 2016

During the summer, I worked with a team of 5 to create a platform for people to teach and learn Linux courses. The platform would let instructors create interactive Linux courses, and let students log in and complete tasks set by the instructors through the web terminal that we integrated. Click here to learn more, or check out the Github page!

The Prophet

January, 2016

For "Build18," my team built a weather data collector using a customly designed PCB, a Particle Photon board as well as other data collector modules. The device would collect local weather data and push them to a database, from which our website would read the data.

Tartan Tag

September, 2015

During the "HackCMU" hackathon, my team built an Android game called "Tartan Tag" modeled after the game "Lazer Tag." We used an API that scanned barcodes, and each scan of a barcode of the other team would win the current team points.

Club Website

July, 2014

I made a website for my high school English club using HTML, CSS, Javascript and bootstrap.
